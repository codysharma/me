Spaceman information

  • Category: Front End
  • Tech Used: Javascript, CSS, HTML, JSON API integration
  • Project URL: https://codysharma.github.io/SpacemanGame/

Description

An update on the hangman word guessing game, this in-browser game can be played either by 2 players alternative or solo with a randomally generated word pulled via API.

Features of this program:

  • Random word pulled via API for playing against the computer
  • An option for playing with or without the timer
  • A choice of how many mistaken letter guesses to allow
  • A hint button appearing after 2 consecutive mistaken guesses
  • A reset button at the end of each round regardless of victory to play again with the same conditions regarding timer, number of mistakes, etc.
  • A checker to prevent spaces or non-alpha cahracters when a player inputs the mystery word
  • Dynamic content and graphics based on stimuli
